# 🚀 Personal Assistant v1.0

Персональний помічник — це консольний додаток на Python для зручного керування контактами та нотатками. Проект розроблений з використанням об'єктно-орієнтованого програмування (ООП) та забезпечує надійне збереження даних.

## 📋 Основні можливості

- **Керування контактами:** Додавання імен, номерів телефонів (з валідацією), електронних адрес та дат народження.
- **Система нотаток:** Створення текстових нотаток з використанням тегів для швидкого пошуку.
- **Розумний пошук:** Можливість знаходити контакти за ім'ям або номером телефону.
- **Нагадування:** Відстеження днів народження на найближчі дні.
- **Автовиправлення:** Програма підказує правильну команду, якщо ви зробили друкарську помилку.
- **Збереження даних:** Автоматична серіалізація даних у бінарні файли при виході.

## 🛠 Технологічний стек

- **Мова:** Python 3.10+
- **Збереження даних:** Модуль `pickle` (серіалізація об'єктів).
- **Валідація:** Регулярні вирази (`re`) для перевірки пошти та номерів.
- **Інтерфейс:** Консольний (CLI).

## 📂 Структура проекту

- `main.py` — точка входу, логіка взаємодії з користувачем.
- `address_book.py` — логіка контейнера для зберігання контактів.
- `record.py` — класи для окремих записів та валідації полів.
- `notes.py` — логіка створення та пошуку нотаток.
- `storage.py` — механізм збереження та завантаження даних (папка `/data`).
- `utils.py` — допоміжні функції (автопідказки команд).
